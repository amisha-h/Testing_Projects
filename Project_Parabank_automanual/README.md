# Project 2: Parabank Web UI Automation


This project focuses on functional test automation for the Parabank demo banking application. It validates critical end-to-end user flows, ensuring system stability for high-value transactions.

# Key Features Automated

The automation script (`test1_bank.py`) successfully handles the following workflows:

1.  User Registration:Automating the creation of a new, valid user account.
2.  Fund Transfer:Performing a fund transfer between accounts and validating the success message on the subsequent         page.
3.  End-to-End Flow: Logging in with new credentials and performing the transfer, simulating a complete user journey.
